Site Goals

This site is intended to show a verity of Front-End/UI skills that I have developed over the last five years. My goal for this site is demonstrate a verity of skills rather than going into great detail in one or two. This site was started in early December of 2018 and is a work in progress so a number of pages in the site will come up as coming soon pages. Thanks you for taking the time to check out my development skills.

Testing

This site has had limited testing with desktop and tablet environments but is not designed to support mobile phones. The testing that has been done on this site should be sufficient to find most issue but due to its limited scope I would not consider it production ready code. It has been tested in Chrome Version 70.0.3538.102, Edge Version 42.17134.1.0, and Firefox Version 63.0.3.
